Amexio Demo App Details

Demo app focuses on the showcase of Amexio UI Components and Features. Here we have created simple app with different pages which is using 54 different amexio components,9 charts, 2 dashboards and 1 map based on Angular4 and Bootstrap4.

  • Showcasing live Amexio charts.
  • Email page showing list of emails with datagrid, tree showing all the nested labels required for showing menus
  • Employee Registration showing two different forms in tabs.
  • Issue Status showing stats using progress bar, customer feedback using carousel view.
  • There are two reports which uses datagrid features like filtering and grouping.
  • Demo App Git Source Code is available on GIT public repository.

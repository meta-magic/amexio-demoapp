Demo app focuses on the showcase of Amexio UI Components and Features.
Here we have created simple app with different pages which is using 50 different amexio components based on Angular4 and Bootstrap4.

Demo app page info

  • Dashboard page showing stats using progress bar, customer feedback using carousel view.
  • Email page showing list of emails with datagrid, tree showing all the nested labels required for showing menus
  • Employee Registration showing two different forms in tabs.
  • There are two reports which uses datagrid features like filtering and grouping.
  • Demo App source code is available on GIT public repository.
